Anthropic Splits Agent SDK Billing June 15

techtimes.com

Starting June 15, Claude Agent SDK, claude -p, and Claude Code GitHub Actions stop drawing from the chat subscription pool. Usage instead consumes a separate metered credit — $20 to $200 per month depending on plan — billed at full API rates with no rollover.

$20–$200/mo, no rollover
POST
WHY IT MATTERS

Audit every cron-based Clay/Claude agent before June 15 — claim the one-time credit, then build a usage dashboard per workflow. High-volume crons like enrichment loops and nightly scoring will hit the cap fast without overflow billing enabled.
