AWS AgentCore: Agents Pay for Data Autonomously

aws.amazon.com

Amazon Bedrock AgentCore launched a preview of agent-native payment capabilities built with Coinbase and Stripe. When an AI agent encounters a paid resource and receives an HTTP 402 response, AgentCore handles the full x402 protocol — wallet auth, stablecoin payment, and proof delivery — without interrupting the agent's reasoning loop. Initial use cases target micropayments between $0.10 and $3.00 for premium data APIs and paywalled content.

$0.10–$3.00 micropayments live
POST
WHY IT MATTERS

The x402 protocol is worth understanding technically: agents encounter a paywall, handle auth, pay in stablecoin, receive proof, and continue — zero human touchpoints. For enrichment workflows this unlocks pay-per-query data APIs that were previously blocked by billing friction. The commerce layer for agentic GTM just arrived.

Meta Ads CLI Brings Campaigns to Command Line

ppc.land

Meta shipped an Ads CLI — a command-line tool wrapping the entire Marketing API into a single installable executable — alongside a companion MCP server. GTM teams can now create, edit, and analyze Meta campaigns directly from a terminal, CI/CD pipeline, or AI agent without custom API wrappers. The tool supports table, JSON, and plain output formats with environment-variable-based auth, and installs via pip or uv.

pip install meta-ads-mcp
POST
WHY IT MATTERS

The table and JSON output format is the GTM engineer's unlock: pipe Meta campaign performance directly into Claude for analysis or into n8n for automated budget reallocation. Build the audit workflow before the optimization workflow — understand what the agent sees before you let it act. Last major ad platform joining the agentic stack.
